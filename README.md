# VolunteerHub

## Tech Stack
- **Frontend:** React (Vite) 
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **Realtime:** Socket.io
- **Deployment:** Docker Compose
  
## Hướng dẫn cách chạy web

B1: Mở Docker desktop.

B2: Chạy lệnh docker compose up --build, vào localhost::5173 (hoặc 5174 tùy) để mở frontend.

## Note: 
Sau này đừng xóa những file docker đi nhé.
